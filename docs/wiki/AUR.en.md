# AUR & Automation (English)

Back: [Wiki Home (EN)](Home.en.md)

## Package

- Package name: `arch-cache-cleaner-git`
- Packaging files: `PKGBUILD`, `.SRCINFO`, `LICENSE`

## `pkgrel` policy

- Bump `pkgrel` for packaging changes
- Keep `pkgver=r0.0` for `-git`; actual version is generated by `pkgver()`

## Local Sync

```bash
./scripts/sync_aur.sh
```

Important: do not run as root.

## Dry Sync without push

```bash
PUSH_CHANGES=0 AUR_DIR=~/Aurstuff/arch-cache-cleaner-git ./scripts/sync_aur.sh
```

## GitHub Automation

Workflow: `.github/workflows/aur-sync.yml`

Required secret:

- `AUR_SSH_PRIVATE_KEY`

Workflow behavior:

1. creates non-root build user
2. regenerates `.SRCINFO`
3. syncs `PKGBUILD/.SRCINFO/LICENSE`
4. pushes only on changes
